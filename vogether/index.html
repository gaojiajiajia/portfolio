VogethR

OVERVIEW
Welcome to VogetheR! We connect VR arcades to VR gamers. Search, find, and book the perfect space for every occasion. Whether you're a player looking to get in a game or an organizer hosting a meetup, we'll play Vogether!
ROLE
UI/UX Designer
PROBLEM
  • Players are excited about VR games but only a few can afford the cost of VR headsets, the computers and other equipment they’ll need to to run high-end VR experiences.
  • An immersive enviroment can only be offered in a dedicated venue, so more and more players are going the VR arcades instead.
  • VR arcades are popping up, yet the modern VR Arcade industry is still a baby industry. The convenience of booking VR arcades reservations online and looking for other players to play multiplayer games are still challenging for most players.
SOLUTION
I design this app that focuses on the booking system and games event. An easy - to - use book workflow is really helpful. Also, the needs of getting the information of VR arcades around gamers can be met.

USER TESTING
I ran a user testing to see if the low-fidelity wireframes work for users
Result
  • no specific button leading the testers to the next step on home page
  • Testers hope to see address of the arcade on confirm page
  • The call option on confirm page is abrupt
Fix
  • Add a go button on homepage
  • Put the address on confirm page
  • Arrenge the call option with the address
Result
  • Users also tend to use search bar to look for the event they want to go.
  • Testers need the info of the venue on the event info page.
  • Liked option wanted on the page of the event.
Fix
  • Add a pop up screen for the arcade info on event page.
  • Add liked icon on event page.
Result
  • Some testers prefer to create the event on the arcade info page instead of typing in the name of the arcade on the other single page.
  • Add to my calendar option on the created event page
Fix
  • Add a “Create an event” option on arcade info page
